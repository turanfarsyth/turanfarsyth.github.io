MyWebApp

World Languages WebApp

A Data-driven Interactive Map for World Language Structures

I am developing this project to create an interactive map that provides a visual interface to interact and view the spread of languages throughout the world. The WALS project is a database of 2679 languages and geographical/informational details on them. There is an active challenge from WALS to create a structured interactive map that presents the data in useful ways.
